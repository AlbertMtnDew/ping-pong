# ping-pong
В репозитории находится прототип мультиплеерной игры Ping-Pong.
Игра реализована на языке программирования Python и использует библиотеку PyGame.
Первый игрок управялет левой ракеткой, второй игрок управляет правой ракеткой, мяч отскакивает от ракеток, прогрывает тот, кто пропустил мяч.
